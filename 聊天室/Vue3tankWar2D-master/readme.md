
### 坦克大战

#### 技术栈 Vue3 Canvas 

***

##### 游戏操作
上下左右键移动 空格开火 吃Starbuff变为无敌状态20s

##### 胜利条件
消灭所有敌方坦克进入下一关

##### 游戏运行
npm install && npm run serve
