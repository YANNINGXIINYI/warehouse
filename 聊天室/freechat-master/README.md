# freechat
nodejs 的一个聊天软件，类似微信。
- 模仿微信的界面，做到简单明了地聊天。
- 无需注册，登录一次后，该电脑将记住用户名，下次直接进入聊天。旨在最简化聊天。
- 支持聊天室，陌生人聊天。

## 软件框架
- node webkit. 桌面客户端的寄宿环境。
- socket.io 客户端与服务器进行通讯。
- html5, css3 , js 客户端开发语言。
